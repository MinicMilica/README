## About The Project
This application was created for easier passenger records. With it we can create, delete and update profiles. Profiles contain basic information about the passenger:
* First and last name
* Passport ID
* Email address

### Built With
This appication is creatied with:
* [Postman](https://getbootstrap.com)
* [SQL 2019](https://jquery.com)
* [Vusal Studio](https://laravel.com)
* [Visual Code](https://getbootstrap.com)
* [Angular 10](https://jquery.com)
* Web API
